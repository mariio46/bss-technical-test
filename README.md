# Technical Test by [BSS Parking](https://bssparking.com/).

## Installation

```bash
git clone https://github.com/mariio46/bss-technical-test.git
```

```bash
cd bss-technical-test
```

**This action is required** if the `.env.local` / `.env.production` does'nt exists

```bash
cp .env.example .env.local
```

All `.env` value i will provide on dm

```bash
npm install
```

```bash
npm run dev
```
